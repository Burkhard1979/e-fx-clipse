Configure as external SDK Windows Users or Preview SDK zips

In cases where the JavaFX SDK is installed not as part of the JDK (e.g. on windows even with JDK-7) or if you want to use a preview SDK provided as zip or through an extra installer

1.

Open Window > Preferences

2.

Open "JavaFX" preference page

3.

Select browse button and navigate to the SDK-Root-Directory (e.g. C:/Program Files (x86)/Oracle/JavaFX 2.1 SDK for the win32 32bit install).
Make sure the SDK matches the bitness of the JDK Eclipse is running in!

4.

Select Ok. You are now ready to create your first JavaFX application

Configure as part of JDK (since 0.0.14 only) MacOS-X users with JDK-7

For installations where the JavaFX-Artifacts are installed in the JDK-Directory structure this easy setup option can be used. Currently this option is only available to Mac-OS-X users who use >= JDK-7u4

1.

Open Eclipse > Preferences ...

2.

Ensure that JDK Built-in is selected (which should be the default for Mac)
