Create an e4 application

Applications based on the e4 platform can use JavaFX instead of SWT as a rendering technology. This tutorial will guide you through the creation of such an application.

1.

Setup a target platform for your application development:

  • Open Preferences > Plugin-in Development > Target Platform
  • Click Add ...
  • Select "Current Target: Copy settings from current target platform".
  • Add ...
  • Select Software Site
  • Add http://downloads.efxclipse.org/p2-repos/releases/latest/ as a site.
  • Select "e(fx)clipse - Composite - FX Runtime - (do not install)" and uncheck "Include required software"
  • Finish and activate the new target.

2.

Run the New > Project ... JavaFX/JavaFX Eclipse 4 Application wizard. Choose a name for your project and leave the rest of the settings on their defaults.

3.

Open the product file of your newly created plug-in project and click on "Launch an Eclipse application".

Note for OS X: Eclipse PDE thinks it is clever and adds some extra flags when launching, but this leads to problems when used with JavaFX.
Open the launch configuration after the first start and:

  • make program arguments hold only: -nl ${target.nl} -consoleLog -nosplash
  • make VM arguments hold only: -Xms40m -Xmx512m -Dosgi.framework.extensions=at.bestsolution.efxclipse.runtime.osgi
